# Computer Vision Portfolio
A collection of my computer vision projects, including vaious image segemntation, thresholding, and feature extraction algorithms, implemented from scratch for deeper mathematical understanding, parameters sensitivity, and analysis of algorithmic behaviour.

## 1- Image Segmentation and Thresholding Studio
A computer vision tool for assessing various image segmentation algorithms, such as clustering (K-Means, Mean Shift, Agglomerative) and thresholding (Otsu, spectral) techniques.

🔗 [Github Repository](https://github.com/Ahmed-Hajhamed/Image-Segmentation-and-Thresholding-Studio)

---
## 2- Image Feature Descriptors and matching Studio
A tool for image feature description algirithms, including SIFT, Harris Corner, and Lambda Minus |R|, in addition to custom similarity metrics, specifically Sum of Squared Differences (SSD) and Normalized Cross-Correlation (NCC), to compute and evaluate feature correspondence.

🔗 [Github Repository](https://github.com/Ahmed-Hajhamed/Image-Feature-Descriptors-Techniques)

---
## 3- Edge and Shape Detection Factory
Implementation of Hough line, circle, and ellipse detection with a GUI for parameter manipulation, such as thresholds and minimum and maximum radii.

🔗 [Github Repository](https://github.com/Ahmed-Hajhamed/Edge-and-Shape-Detection-Factory)
