# Computer Vision Portfolio
A collection of my computer vision projects, including various image segmentation, thresholding, and feature extraction algorithms implemented from scratch for deeper mathematical understanding, parameter sensitivity analysis, and exploration of algorithmic behavior.

## 1- Image Segmentation and Thresholding Studio
A computer vision tool for exploring various image segmentation algorithms, such as clustering methods (K-Means, Mean Shift, and Agglomerative Clustering) and thresholding techniques (Otsu and Spectral Thresholding).

🔗 [GitHub Repository](https://github.com/Ahmed-Hajhamed/Image-Segmentation-and-Thresholding-Studio)

---

## 2- Image Feature Descriptors and Matching Studio
A tool for image feature description and matching algorithms, including SIFT, Harris Corner Detection, and Lambda Minus |R|, in addition to custom similarity metrics such as Sum of Squared Differences (SSD) and Normalized Cross-Correlation (NCC) for computing and evaluating feature correspondences.

🔗 [GitHub Repository](https://github.com/Ahmed-Hajhamed/Image-Feature-Descriptors-Techniques)

---

## 3- Edge and Shape Detection Factory
Implementation of Hough line, circle, and ellipse detection algorithms with a GUI for parameter manipulation, including thresholds and minimum/maximum radii adjustment.

🔗 [GitHub Repository](https://github.com/Ahmed-Hajhamed/Edge-and-Shape-Detection-Factory)
